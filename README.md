# image-processing-api

## Scripts to run the project

---

to run tests:

```
yarn test
```

to start the project:

```
yarn dev
```

to build the project:

```
yarn build
```

## Endpoints to test

---

- Base URL: <http://localhost:5000/api>

### Products

- index: /products
- show: /products/{product_id}
- create: /products/create

### Orders

- show: /orders/{user_id}

### Users

- index: /users
- show: /users/{user_id}
- create: /users/create
